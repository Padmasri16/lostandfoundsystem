# Lost and Found Management System

## Project Overview
The Lost and Found Management System is a web-based application designed to help users report lost items, register found items, and recover misplaced belongings efficiently.

## Project Methodology
This project follows Agile Scrum methodology and is managed using Jira Software. Development is carried out in sprints with proper backlog planning, issue tracking, and automation support.

## Features
- Report lost items
- Report found items
- View lost and found item listings
- Match lost items with found items
- Bug tracking and testing using Jira
- Automation using GitHub and Jira integration

## Tools Used
- Visual Studio Code
- GitHub
- Jira Software (Scrum)
- GitHub Actions (CI automation)

## Automation
Automation is implemented to track code changes, run build workflows, and update Jira issues with development and testing status.
